# @aycd/editor

Plate.js-based rich text editor for AYCD.

## Status

**Stub implementation** - Full Plate.js integration will be added in future iterations.

## Planned Features

- Markdown support
- Block-based editing
- Collaborative editing hooks
- Custom slash commands
- AI integration points
- Export/import formats

## Usage

```tsx
import { PlateEditor } from '@aycd/editor';

<PlateEditor
  content={documentContent}
  onChange={handleChange}
  readOnly={false}
/>
```
